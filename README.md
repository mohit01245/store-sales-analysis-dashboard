# 📊 Store Sales Analysis Dashboard (Excel)

## 📌 Project Overview
This project is an interactive Microsoft Excel dashboard designed to analyze store sales performance. It provides valuable business insights through Pivot Tables, Pivot Charts, Slicers, and data visualization techniques, helping users understand sales trends, customer behavior, and overall business performance.

---

## 🎯 Objective
The objective of this project is to transform raw sales data into meaningful insights using Microsoft Excel, enabling better business decision-making through interactive dashboards.

---

## ✨ Features
- Interactive Sales Dashboard
- Monthly Sales & Orders Analysis
- Sales by Gender
- Order Status Analysis
- Top 5 Performing States
- Age Group vs Gender Analysis
- Sales Channel Distribution
- Dynamic Slicers for Filtering
- Professional Dashboard Design

---

## 🛠️ Tools & Technologies

## 📈 Dashboard Insights
- Analyze monthly sales and order trends.
- Compare purchases by men and women.
- Track delivered, cancelled, returned, and refunded orders.
- Identify the top-performing states.
- Analyze customer age groups.
- View sales distribution across different sales channels.
